5주차

1. Simulation Pool
Simulation 객체 미리 만들어 놓고 재사용


2. 진동문제
- 단일 Mass모드에서는 각 spring의 길이를 고정하여 탄성력 삭제

- 상기의 방법을 다중 Mass에 적용 시
 Spring 길이 고정에 따른 Mass위치 강제 변경
 해당 위치 변경이 또 다른 Mass에 영향을 주면서 시물레이션이 망가짐. 
-> 다중 Mass모드에서는 탄성 적용 


3. 크로스 헤어 그리기
- 직교투영과 기하셰이더로 렌더링


4. 체공 상태에서 방향키 입력시, 이동값이 아닌 속도값 제어 
- 방향키로 Vel을 더해주자, 대신 내가 입력하는 방향쪽으로 이미 일정이상의 속도면 증가x


5. 충돌 판정
- 로프의 지면 충돌 + 쓸림 + 튕김 
- 캐릭터 스윙 중 지면 충돌 시 -> 스윙 취소
- 캐릭터 스윙 중 오브젝트 충돌 시 -> 스윙 취소, 캐릭터 Vel초기화, 
- 캐릭터 체공 중 오브젝트 충돌 시 -> 캐릭터 Vel초기화
문제점 : 
캐릭터가 오브젝트와 옆이나 위로 충돌시, 해당 방법도 자연스럽지만.
아래와 충돌시, 오브젝트를 올라 타야함.
